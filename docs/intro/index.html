    通用说明

    签名算法#

    1. 将请求参数(排除sign参数)按Ascii码序进行排序并拼接成字符串。
    2. 将发放的apiSecret拼接在最后。
    3. 将拼接后的字符串进行sha256加密, 转成小写。
    示例#
    需要请求的参数和得到的apiSecret#

    apiSecret="abcdefg"

    {  "appid": "test",  "merchantTradeNumber": "1583308458233",  "amount": 100,  "timestamp":1628660966207,  "notifyUrl":"http://*****/"
}

    注意： -- null值不参与签名

    拼接请求参数#

    "amount=100&appid=test&merchantTradeNumber=1583308458233..."

    拼接apiSecret#

    String str="amount=100&appid=test&merchantTradeNumber=1583308458233&此处为secret"

    进行SHA256加密#

    -- String sign = DigestUtil.sha256Hex(str) "309403f3d530eaad5f716e88539fb7fea94b954643cd9bdb078938c27602e6ea"
