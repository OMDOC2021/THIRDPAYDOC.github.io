    通用说明

    Ocean Pay接入流程：

    1. 商户先向Ocean Pay申请入网商户，提供:手机号码,邮箱,联系人名3项信息；
    2. 运营人员为商户创建测试账号，提供给商户测试账号及密码；
    3. 商户获得测试账号和密码后，登陆支付通道后台：地址 ，在商户信息页里可以找到分配的：“appid” 和 “appsecret”；
    4. 商户根据Ocean Pay提供的API文档,进行H5支付的开发和接入: https://omdoc2021.github.io/THIRDPAYDOC/docs/intro；
    5. 商户可以通过接口查询支付结果,或在后台(地址 )的交易列表进行查看；

    签名算法#

    1. 将请求参数(排除sign参数)按Ascii码序进行排序并拼接成字符串。
    2. 将发放的apiSecret拼接在最后。
    3. 将拼接后的字符串进行sha256加密, 转成小写。
    示例#
    需要请求的参数和得到的apiSecret#

    apiSecret="abcdefg"

    注意： -- null值不参与签名

    "amount=100&appid=test&merchantTradeNumber=1583308458233..."

    拼接apiSecret#

    String str="amount=100&appid=test&merchantTradeNumber=1583308458233&此处为secret"
