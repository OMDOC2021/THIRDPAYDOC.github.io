    交互流程

    使用app 获取下单前参数

    向我方获取请求参数

    请求URL#

    /api/wechat/app

    请求方式#

    POST请求，Content-Type为application/json

    请求字段#

    字段类型长度示例可否为空说明
    appidstring8-32"test1234"否商户接入appid，需创建商户申请
    merchantTradeNumberstring1-64---否商户方单号
    amountLong0-12---否金额(THB),精确到分，例如1THB,需传100
    timestampLong------否下单时间戳，10分钟内下单有效
    notifyUrlstring12-20---否异步回调地址
    openidString------否微信用户openid
    paymentMethodIdint------否固定值：（正式环境为1，测试环境为2）代表微信
    payMethodString------否固定: APP
    orderInfostring------否商品信息，展示给买家的信息
    storeIdString------否店铺id
    signstring12-20---否签名串

    请求示例#

    {  "appid":"dp9m9hl1wx563wbk",  "merchantTradeNumber":"2022110516340000000000039",  "amount":"100",  "timestamp":"1649933895000",  "notifyUrl":"http://baidu.com",  "sign":"5d04e4679e224daaabc4630e1d24c14cbc0fc46efb334d4ffc7a53047ff0ab1a",  "paymentMethodId":"1",  "storeId":"1",  "payMethod":"APP",  "orderInfo":"test",  "wechatAppid":"wx8c23b713feedf361"}

    返回内容说明#

    字段类型示例说明
    codeint200业务码 200 为成功下单
    msgstring"ok"对业务码说明
    dataobject-----
    merchant_idstring----商户号
    merchant_order_idstring----我方订单号
    order_amountstring----订单金额
    order_currencystring----币种
    order_infostring----商品展示信息
    product_codestring----产品CODE:WECHAT_PAY
    payment_methodstring----支付类型:APP_PAYMENT
    appidstring----APP ID
    customerobject----商户端客户信息
    notify_urlstring----回调地址
    redirect_urlstring----支付完成展示页面
    signstring----签名串

    customer#

    字段类型示例说明
    merchant_user_idstring----商户端用户ID,唯一
    full_namestring----商户端客户名称

    返回内容示例#

    {  "msg": "ok",  "code": 200,  "data": {    "merchant_id": "142021081000282006",    "merchant_order_id": "1514559058760302592",    "order_amount": "1.00",    "order_currency": "THB",    "order_info": "OFFLINE",    "product_code": "WECHAT_PAY",    "payment_method": "INAPP_PAYMENT",    "appid": "wx8c23b713feedf361",    "customer": "CustomerVO(merchant_user_id=1514559058760302592, customer_type=null, first_name=null, last_name=null, full_name=1514559058760302592)",    "notify_url": "https://test-payway.cathayinter.com/lianlian/api/notify",    "redirect_url": "",    "sign": "HZk/XdrNo2mosSxi/ojpJ943gdjPMEbZfF3zZ9TGr+Z8Qr03tGtfybBXBBkrPAkpx3OnkJulOoojCJ3LtzDPiX7gHNKSZgNwC6kE6J1XusIeD9/5WKMdMfKMWipTAp21VILTKFZSmQMcjdhTuQES+dhw5/aR2ilEGPgNYM5qUik="  }}
