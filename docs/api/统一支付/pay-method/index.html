    获取支付方式

    请求URL#

    /api/methods

    请求方式#

    GET

    请求字段#

    字段类型长度示例可否为空说明
    appidstring8-32"test1234"否商户接入appid，需创建商户申请

    请求示例#

    https://test-payway.cathayinter.com/api/methods?appid=****

    返回内容说明#

    字段类型长度示例可否为空说明
    codeint---200否业务码 200 为成功下单
    msgstring---"ok"否对业务码说明
    datadata-----否对象参数

    data参数说明#

    字段类型长度示例可否为空说明
    namestring---"alipay"否支付方式名称
    methodIdlong---1否支付方式ID,下单时需传递
    weightstring---"10"否权重

    返回内容示例#

    {    "msg": "ok",    "code": 200,    "data": [        {            "methodId": 1,            "name": "alipay",            "weight": 10        },        {          "methodId": 2,            "name": "promptpay",            "weight": 10        },        {          "methodId": 3,            "name": "ktbcard",            "weight": 10        },        {          "methodId": 4,            "name": "wechat",            "weight": 10        }    ]}
