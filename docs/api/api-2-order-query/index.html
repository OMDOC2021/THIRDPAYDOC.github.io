    订单信息查询

    请求URL#

    /api/order/query/{appid}/{merchantNo}

    请求方式#

    GET请求, 每秒1次

    请求字段#

    字段类型长度示例可否为空说明
    appidstring8-32"test1234"否商户接入appid，需创建商户申请
    merchantNostring12-20---否商户方单号

    请求示例#

    /api/order/query/test/13912381838138183

    返回内容说明#

    字段类型长度示例可否为空说明
    codeint---200否业务码 200 为成功下单
    msgstring---"ok"否对业务码说明
    dataobject-----否订单详情

    data 说明#

    字段类型说明
    idlong订单序号
    createTimelong创建订单时间戳
    merchantNameString商户名称
    finishedTimelong订单完结时间
    statusString订单状态
    amountString下单金额，精确到分
    currencyString币种THB
    paymentTypeString支付类型
    tradeNumberString我方单号
    merchantTradeNumberString商户方单号
    feeAmountString手续费
    receivableAmountString到账金额
    feeRateString费率
    remarkString备注
