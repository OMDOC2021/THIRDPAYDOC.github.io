    异步回调 自动触发

    注意#

    请以主动查询为主

    请求URL#

    接入方下单时，提供的notifyUrl信息。

    请求方式#

    POST请求, 1次

    请求字段#

    字段类型说明
    idlong订单序号
    createTimelong创建订单时间戳
    merchantNameString商户名称
    finishedTimelong订单完结时间
    statusString订单状态(SUCCESS(完成), FAIL(失败), INIT(下单中) ,PAYING(等待支付), CLEARING(已清算))
    amountString下单金额，精确到分
    currencyString币种THB
    paymentTypeString支付类型
    tradeNumberString我方单号
    merchantTradeNumberString商户方单号
    feeAmountString手续费
    receivableAmountString到账金额
    feeRateString费率
    signString签名
